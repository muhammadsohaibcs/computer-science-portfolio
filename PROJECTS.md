### 2️⃣ Secure Drone Command Authorization System

<div align="center">

**Zero-trust cryptographic control system for drones and robots**

![Tech](https://img.shields.io/badge/Python-blue?logo=python&logoColor=white) ![Tech](https://img.shields.io/badge/Cryptography-red?logo=crypto&logoColor=white) ![Tech](https://img.shields.io/badge/Blockchain-orange?logo=blockchain&logoColor=white)

</div>

**Description:**
A high-security command-and-control system designed for drones, robots, and autonomous systems. Implements a zero-trust architecture ensuring that every command is authenticated, authorized, encrypted, and verified. Features blockchain-based audit logging, replay attack protection, role-based access control, and multi-layer cryptographic security.

**Key Technologies:**
```
Python, AES-256-GCM, Ed25519, PKI, Blockchain, Zero-Trust Architecture, Cryptography
```

**Highlights:**
- ✅ Zero-trust architecture for command validation
- ✅ End-to-end encrypted communication (AES-256-GCM)
- ✅ Digital signatures using Ed25519
- ✅ Blockchain-based immutable audit trail
- ✅ Replay attack protection using nonces & timestamps
- ✅ Role-based and policy-based authorization
- ✅ Real-time command verification

**Directory:** [information-security](information-security)
